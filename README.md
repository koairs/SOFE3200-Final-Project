# SOFE3200-Final-Project
User Account Management Script
-------------------------------------

## Requirements
- Linux OS
- Root access (sudo)
- Bash shell

## Setup
Download all project files into a folder, then run:

chmod +x setup.sh
./setup.sh

## Running the Script
Use:

sudo ./user_manager.sh

## Features
- Add user
- Delete user
- Modify user (password, shell, fullname, group changes)
- List users
- Error handling
- Logging in ./logs/script.log

## Logs
All actions are logged in:

logs/script.log
